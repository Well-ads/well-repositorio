Linguagem Popularidade Tipos de Uso Facilidade de Aprendizado Opinião Pessoal
JavaScript Muito Alta Web (frontend e backend), aplicações web interativas Moderada Ótima para web development e projetos de pequeno a médio porte.
Java Alta Aplicações de desktop, desenvolvimento Android, servidores Moderada Versátil e robusta, mas pode ser verbosa (implica dizer que é mais prolixa e requer mais código para fazer a mesma coisa em comparação com uma linguagem mais concisa).
PHP Alta Desenvolvimento web (backend) Fácil Ótima para web, mas requer atenção à segurança.
Python Muito Alta Desenvolvimento web, análise de dados, IA, automação Fácil Fácil de aprender, ótima para iniciantes. Particularmente é a minha preferida, pois foi a primeira que conheci. Sintaxe fácil, por isso o aprendizado é rápido.
Ruby Baixa Desenvolvimento web e automação Fácil Elegante e expressiva, mas menos popular.
C# Alta Desenvolvimento de aplicativos Windows, jogos, aplicações empresariais Moderada Excelente para desenvolvimento Windows e jogos.
C++ Alta Desenvolvimento de sistemas, jogos, aplicações de alto desempenho Moderada Poderosa, mas complexa. Excelente para sistemas de alto desempenho. Achei Complexa, de difício aprendizado, maior risco de erros de programação, código extenso em comparação com linguagens de alto nível mais modernas.
Swift Média Desenvolvimento de aplicativos iOS e macOS Moderada Moderna e eficiente para desenvolvimento Apple. Por ser projetada principalmente para o ecossistema Apple, achei bem limitada.
